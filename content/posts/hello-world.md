## 소개

안녕하세요! 이 블로그는 **Hugo** 정적 사이트 생성기와 **GitHub Pages**를 사용하여 만든 기술 블로그입니다.

## 기술 스택

이 블로그는 다음 기술들로 구성되어 있습니다:

- **Hugo** — 빠른 정적 사이트 생성기
- **PaperMod** — 깔끔한 기술 블로그 테마
- **GitHub Pages** — 무료 정적 사이트 호스팅
- **GitHub Actions** — 자동 빌드 및 배포

## 코드 하이라이팅 테스트

Hugo는 기본적으로 코드 하이라이팅을 지원합니다:

```python
def hello():
    print("Hello, World!")

if __name__ == "__main__":
    hello()
```

```javascript
const greet = (name) => {
  console.log(`Hello, ${name}!`);
};

greet("World");
```

## SEO 최적화

이 블로그에는 다음과 같은 SEO 최적화가 적용되어 있습니다:

1. **구조화 데이터** — JSON-LD (Article/BlogPosting)
2. **Open Graph 태그** — 소셜 미디어 공유 최적화
3. **Sitemap** — 검색 엔진 크롤링 지원
4. **Robots.txt** — 크롤러 가이드
