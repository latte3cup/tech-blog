## Claude란?

Claude는 Anthropic이 만든 AI 어시스턴트입니다. 코딩, 분석, 글쓰기 등 다양한 작업에 활용할 수 있으며, 특히 긴 코드 분석과 정확한 지시 따르기에 강점이 있습니다.

### Claude 모델 종류

| 모델 | 특징 | 용도 |
|------|------|------|
| **Opus** | 최고 성능, 복잡한 추론 | 아키텍처 설계, 복잡한 디버깅 |
| **Sonnet** | 성능/속도 균형 | 일반 코딩, 코드 리뷰 |
| **Haiku** | 가장 빠름 | 간단한 질문, 코드 자동완성 |

## 코딩에 Claude를 쓰는 방법

### 1. 웹 채팅 (claude.ai)

가장 기본적인 방법입니다. 브라우저에서 바로 사용합니다.

- 코드 파일을 붙여넣기해서 질문
- 에러 메시지 복사 → 해결법 요청
- 새 기능의 코드 생성 요청

### 2. Claude Code (CLI 도구)

터미널에서 직접 사용하는 AI 코딩 에이전트입니다. 파일을 읽고, 수정하고, 명령어를 실행할 수 있습니다.

```bash
# 설치
npm install -g @anthropic-ai/claude-code

# 프로젝트 디렉토리에서 실행
cd my-project
claude

# 특정 작업 요청
claude "이 프로젝트의 테스트를 실행하고 실패하는 것 수정해줘"
```

Claude Code의 장점은 **프로젝트 컨텍스트를 자동으로 이해**한다는 것입니다. 파일 구조, 의존성, 기존 코드 패턴을 파악해서 일관된 코드를 생성합니다.

### 3. API 활용

자동화 파이프라인이나 자체 도구에 Claude를 통합할 수 있습니다.

```python
import anthropic

client = anthropic.Anthropic()

message = client.messages.create(
    model="claude-sonnet-4-20250514",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "Python으로 퀵소트 구현해줘"}
    ]
)

print(message.content[0].text)
```

## 효과적인 프롬프트 작성법

### 구체적으로 요청하기

```

"로그인 만들어줘"

- 에러 처리 포함"
```

## Claude 사용 시 주의점

### 1. 코드 검증은 필수

AI가 생성한 코드를 그대로 사용하지 마세요. 반드시 직접 확인하고 테스트해야 합니다.

- 생성된 코드가 실제로 동작하는지 실행
- 보안 취약점이 없는지 검토
- 기존 코드베이스와 호환되는지 확인

### 2. 민감한 정보 주의

API 키, 비밀번호, 내부 비즈니스 로직 등 민감한 정보를 프롬프트에 포함하지 않도록 주의하세요.

### 3. 최신 정보 한계

AI 모델은 학습 데이터의 시점 이후 정보를 모를 수 있습니다. 최신 라이브러리 버전이나 API 변경사항은 공식 문서를 확인하세요.

## Claude vs 기존 개발 방식

| 작업 | 기존 | Claude 활용 |
|------|------|-------------|
| 보일러플레이트 코드 | 직접 작성 / 복붙 | 즉시 생성 |
| 에러 디버깅 | Stack Overflow 검색 | 에러 메시지 붙여넣기 → 해결 |
| 코드 리뷰 | 동료에게 요청 | 즉시 리뷰 (동료 리뷰 대체는 아님) |
| 새 언어/프레임워크 학습 | 문서 + 튜토리얼 | 예제 코드 요청 + 설명 |
| 정규식 작성 | 구글링 + 시행착오 | 설명하면 바로 생성 |

## 마무리

AI 코딩 도구는 개발자를 대체하는 것이 아니라, 생산성을 높이는 도구입니다. 반복적인 작업은 AI에게 맡기고, 핵심 로직과 아키텍처 결정에 집중하세요. 가장 중요한 것은 AI가 생성한 코드를 이해하고 검증하는 능력입니다.
